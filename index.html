About

Padingo is a web-based digital assistant that will help you find answers to your questions. Made and published by LordPaijo and Rifky1720 at home, on the
16th of November 2024. It uses the WebKit-SpeechRecognition API on JavaScript and was made using Javascript. It's free and open-source and can be obtained from this GitHub repository.

How to use

Make sure your browser supports Webkit-SpeechRecognition API PERFECTLY. This will ensure that features such as voice recognition won't have issues. Otherwise, you will have a bad experience, or worse, it will stop working. After you're sure of everything, you can now use it by simply clicking the speak button. Say anything... It'll google it for you...
